# 큐
![[Pasted image 20240831223939.png]]
한쪽으로 넣고, 반대쪽에서 꺼내는 자료구조.
가장 먼저 넣은 원소가 먼저 나오므로, FIFO 구조이다.

1. 원소의 추가/제거가 O(1)
2. 제일 앞/뒤의 원소 확인 O(1)
3. 앞/뒤 제외 나머지 원소들의 확인/변경이 원칙적으로 불가

보통 *head가 첫번째 원소*, *tail이 마지막 원소 다음 원소*

큐는 BFS나 Flood FIll을 할 때 사용함
